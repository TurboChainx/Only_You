# Public URLs for the privacy policy

Use **one** of these in Google Play Console → App content → Privacy policy.

## Option A — GitHub-rendered page (live after push to `main`)

**Markdown (readable in browser):**  
https://github.com/TurboChainx/Only_You/blob/main/docs/privacy-policy.md

*(Repository must be **public**, or the Play review team must be able to open the URL.)*

## Option B — Static HTML (recommended after enabling GitHub Pages)

1. In the repo: **Settings → Pages → Build and deployment → Source:** `Deploy from a branch`  
2. **Branch:** `main` → **Folder:** `/docs` → Save  

Then use:

**https://turbochainx.github.io/Only_You/privacy-policy.html**

(If your Pages URL differs, check **Settings → Pages** for the exact site URL.)

## Option C — Raw HTML (not ideal for users; OK for testing only)

https://raw.githubusercontent.com/TurboChainx/Only_You/main/docs/privacy-policy.html

---

Replace `privacy@onlyyou.app` in the policy with your real support email when you go live.
